# SupplyPulse 🚢

## Built for Ship with Kiro Hackathon

### Problem
Pakistan's $50 billion trade industry has zero real-time visibility into supply chain disruptions. Businesses find out about Karachi port delays, Shanghai congestion, and Rotterdam strikes hours too late — by then, millions are already lost.

### Solution
SupplyPulse is Pakistan's first real-time supply chain disruption monitoring dashboard. It monitors 147 trade routes, predicts disruptions 7 days in advance using AI, and instantly suggests alternative routes.

### Key Features
- Real-time monitoring of 147 Pakistan trade routes
- AI-powered disruption prediction for next 7 days
- Instant alternative route suggestions
- Live alerts from Karachi, Port Qasim, Gwadar and Lahore Dry Port
- Demo Mode for live presentations
- Auto Debug Monitor using Kiro Agent Hooks

### Built With Kiro
- Used Kiro AI to generate entire React codebase
- Created 2 Agent Hooks: Auto Debug Monitor and Supply Chain Optimizer
- All features built through Kiro prompts

### Tech Stack
- React + Vite
- TailwindCSS
- Chart.js
- Kiro AI
